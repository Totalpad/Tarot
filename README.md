# Tarot Cards

![Screenshot 1](/src/Assets/1.jpg)

## Description

This is a Tarot card reading application that allows users to perform Tarot readings even if they don't have physical cards. The app provides access to a full list of Tarot cards and the option for random readings to get answers to questions and enrich your experience.

## Features

- **Card List**: Browse the complete list of Tarot cards. You can search for a card by name or explore the entire card list.

- **Random Cards**: Get random Tarot card readings. The app will generate three random cards for you, offering insights and guidance for your questions.

## Technologies Used

- **React**: The app is built using React, a popular JavaScript library for building user interfaces.

- **React Router**: React Router is used for handling navigation and creating distinct pages for different sections of the app.

- **CSS**: Cascading Style Sheets are used for styling the app and creating a visually pleasing user interface.

## Getting Started

1. Clone this repository to your local machine.
2. Install the necessary dependencies using `npm install`.
3. Start the development server with `npm start`.

## Usage

- Access the app on your local server.
- Browse the Tarot card list.
- Click "Random Cards" to get three random Tarot card readings.
